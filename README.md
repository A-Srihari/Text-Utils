# TextUtils 

TextUtils is a simple text utility application that allows users to manipulate and analyze text.

==> Features
- Convert text to Uppercase
- Convert text to Lowercase
- Remove Extra Spaces
- Copy text to Clipboard
- Dark and Light Mode Toggle

==> Technologies Used
- HTML5
- CSS3
- Bootstrap
- JavaScript
